## Warp Affine
Warp affine adalah proses memetakan (meng-warp) citra menggunakan transformasi afine, yaitu transformasi geometris yang dapat mencakup, translasi, rotasi, scaling, shearing (pergeseran miring), atau gabungan semuanya
